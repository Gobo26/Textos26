# Textos26

Textos26 és un petit espai de lectura per a textos personals, emocions i fragments de vida.

Web: https://gobo26.github.io/Textos26/

## Contingut

- `index.html`: portada i índex dels textos.
- `textos/despres-de-sant-jordi.html`: peça narrativa publicada dins el quadern.

## Criteri

El projecte prioritza una lectura tranquil·la, una estètica de paper i un to proper. Els textos són originals de l'autor i es mostren per a lectura personal.
